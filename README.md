# JS приложение для поиска фильмов

## Технологии

- ⚛️ React 18
- 📦 Redux Toolkit (управление состоянием)
- 🚦 React Router v6 (навигация)
- 📝 TypeScript (типизация)
- 🎨 SCSS (стилизация)
- 🔧 Webpack (сборка)
- 🎬 Кинопоиск API

## Функциональность

- ✅ Главная страница с приветствием
- ✅ Страница входа (Admin/12345)
- ✅ Страница с популярными фильмами
- ✅ Защищенный маршрут профиля
- ✅ Сохранение авторизации в localStorage
- ✅ Адаптивный дизайн
- ✅ Анимации и современный UI

## Установка и запуск

- Клонировать репозиторий:
git clone https://github.com/oper0upolnomocheniy777/js-simple-page

- Перейти в папку проекта:
cd js-simple-page

- Установить зависимости:
npm install

- Запустить проект:
npm start

## API Кинопоиска
Для работы с фильмами нужно получить API ключ:

- Перейти на https://kinopoiskapiunofficial.tech

- Зарегистрироваться

- Получить ключ

- Вставить его в src/api/moviesApi.ts
